## 1. Qu’est-ce que Python ?

Python est un langage de programmation interprété, simple et polyvalent.  
Il est :

- lisible (syntaxe proche du langage naturel),
- multiplateforme (Windows, Linux, macOS),
- très utilisé en :
  - analyse de données,
  - intelligence artificielle et machine learning,
  - développement web,
  - scripts système,
  - automatisation.

Python met l’accent sur la clarté du code et la productivité du développeur.

## 2. Variables et types de base

### 2.1 Variables

Une variable est un nom qui référence une valeur en mémoire.  
On crée une variable simplement avec l’opérateur "=" :

x = 5
nom = "Alice"
pi = 3.14

Le type est déduit automatiquement.

### 2.2 Types de base

- int : entier
- float : réel
- bool : booléen
- str : chaîne de caractères

Exemples :

age = 27
temperature = 19.5
est_etudiant = True
message = "Hello, Python !"

## 6. Structures de contrôle

### 6.1 Conditions

age = 20
if age < 18:
    print("Mineur")
elif age == 18:
    print("Tout juste majeur")
else:
    print("Majeur")

### 6.2 Boucles for

for i in range(5):
    print(i)

### 6.3 Boucle while

compteur = 0
while compteur < 3:
    print(compteur)
    compteur += 1
